# Quick guide
```
$ make gen       // 정적 페이지 생성
$ make preview   // 미리보기
$ make new-post  // 새로운 글 작성
$ make new-page  // 새로운 페이지 작성
$ make new-slide // 새로운 웹 슬라이드 작성
```
